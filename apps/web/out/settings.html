Importar historial completo

Conecta tu casilla de Gmail para importar automáticamente todos los mails de PedidosYa, Rappi, Uber, Mercado Libre, bancos y más.

¿Cómo configurar Gmail?

  1. Activa "Acceso IMAP" en Gmail → Configuración → Reenvío e IMAP
  2. Genera un App Password en tu cuenta Google → Seguridad → Contraseñas de apps
  3. Agrega IMAP_USER e IMAP_PASSWORD al .env del servidor

Reenvío automático de emails

Para importar resúmenes bancarios automáticamente, configura el reenvío de correos electrónicos:

Gmail

  1. Ir a Configuración → Reenvío y POP/IMAP
  2. Agregar dirección de reenvío: ingest@{TU_DOMINIO}
  3. Crear un filtro: De = alertas@galicia.com.ar OR notificaciones@brubank.com
  4. Acción: Reenviar a la dirección del paso 2

Envío directo vía API

POST http://192.168.1.4:4000/api/ingest/email
Content-Type: application/json

{
  "from": "alertas@galicia.com.ar",
  "subject": "Consumo con tu tarjeta",
  "textBody": "..."
}

Remitentes soportados: Galicia, Brubank, BBVA, Mercado Pago, PayPal, Wise
